# Agriculture Education Website

An ASP.NET Core web application that educates users about different agricultural practices including Livestock Ranching, Commercial Agriculture, and Organic Farming.

## Features

- **Home Page**: Overview of three main agriculture types
- **Livestock Ranching**: Detailed information about animal farming
- **Commercial Agriculture**: Information about large-scale farming
- **Organic Farming**: Details about sustainable farming practices
- **Responsive Design**: Works on desktop and mobile devices
- **External Resources**: Links to authoritative agricultural organizations

## Technologies Used

- ASP.NET Core MVC
- HTML5 & CSS3
- JavaScript
- Bootstrap
- Font Awesome Icons
- Responsive Grid Layout

## Agriculture Types Covered

1. **Livestock Ranching**
   - Cattle farming
   - Sheep and goat farming
   - Extensive land use
   - Meat, wool, and hide production

2. **Commercial Agriculture**
   - Large-scale production
   - Modern technology and machinery
   - Market-oriented approach
   - Global distribution

3. **Organic Farming**
   - Natural processes
   - No synthetic chemicals
   - Biodiversity focus
   - Environmental sustainability

## Getting Started

1. Clone the repository
2. Open the solution in Visual Studio
3. Build the project (Ctrl+Shift+B)
4. Run the application (F5)

## Navigation

- **Home**: Overview of all agriculture types
- **Livestock Ranching**: Animal farming details
- **Commercial Agriculture**: Large-scale farming information
- **Organic Farming**: Sustainable practices
